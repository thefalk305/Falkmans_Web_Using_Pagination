Instructions for adding a new flip-card to the gallery:

  • Click 'Add a New Card'. This opens the New Card Form.
  • Enter the person's name in the 'Name:' field.
  • Enter a brief description for the person. This may be several sentences.
  • Under 'Upload Photo' click 'Choose File'. This will open an explorer window. Navigate to the location of the persons photo, select it and Click 'Open'.
  • Under 'Upload Biography' click 'Choose File'. This will open an explorer window. Navigate to the location of the persons Biography, select it and Click 'Open'.
  • Note: The biography file must be a .txt file. It would be helpful if the filename contained the persons name. (first_last).
  • Click 'Submit to Gallery'. The persons flip-card will be added to the Gallery.
